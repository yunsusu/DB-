# 테이블에 CONSTRAINT 걸기

CONSTRAINT는 제약사항이라는 뜻으로

만약 컬럼에 값이 30,000,000 보다 큰 값이 들어가면 안된다고 제약사항을 걸려면

```jsx
ALTER TABLE 테이블이름
	ADD CONSTRAINT 제약이름 CHECK (registration_number < 30000000);
```

이 제약사항을 삭제하려면

```jsx
ALTER TABLE 테이블이름 DROP CONSTRAINT 제약이름
```

### 두개 이상의 제약이 담긴 제약사항

코드로 보면

```jsx
ALTER TABLE 테이블이름
	ADD CONSTRAINT 제약이름
	CHECK (email LIKE '%@%' AND gender IN ('m','f'));
```
